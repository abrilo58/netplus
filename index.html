Question 1

A network administrator wants to implement an authentication process for temporary access to an organization's network. Which of the following technologies would facilitate this process?

A. Captive portal

B. Enterprise authentication

C. Ad hoc network

D. WPA3

Show Answer & Explanation ▶

Correct Answer: A. Captive portal

Why this is correct: A captive portal is a web page that users must interact with before being granted access to a network. It's commonly used in hotels, airports, and businesses to provide temporary access after users authenticate (often with a password or through social media). This makes it perfect for temporary network access scenarios.

Why others are incorrect:

  • Enterprise authentication: This is for permanent employees using company credentials, not temporary access.
  • Ad hoc network: This refers to device-to-device connections without infrastructure, not authentication.
  • WPA3: This is a wireless security protocol for encryption, not temporary access control.

Beginner Tip: Think of a captive portal like a bouncer at a club - it checks your ID (credentials) before letting you in, but only for that one visit.

Question 2

Which of the following should be used to obtain remote access to a network appliance that has failed to start up properly?

A. Crash cart

B. Jump box

C. Secure Shell

D. Out-of-band management

Show Answer & Explanation ▶

Correct Answer: D. Out-of-band management

Why this is correct: Out-of-band (OOB) management uses a separate, dedicated channel (often through a serial console port) to access network devices when the primary network connection is down. This is crucial when a device won't boot normally, as it doesn't rely on the device's operating system being functional.

Why others are incorrect:

  • Crash cart: This is physical equipment (monitor, keyboard) rolled to the device location - not remote access.
  • Jump box: This is a secured intermediate server for network access, but requires the target device to be operational.
  • Secure Shell (SSH): Requires the device to have network services running, which isn't possible if it failed to start.

Beginner Tip: Imagine OOB management like an emergency phone line that works even when the main power is out - it's always available for critical access.

Question 4

Which of the following best describes what an organization would use port address translation for?

A. VLANs on the perimeter

B. Public address on the perimeter router

C. Non-routable address on the perimeter router

D. Servers on the perimeter

Show Answer & Explanation ▶

Correct Answer: B. Public address on the perimeter router

Why this is correct: Port Address Translation (PAT) is a type of NAT that allows multiple devices to share a single public IP address by using different port numbers. It's typically configured on perimeter routers to conserve public IP addresses and provide security through obscurity.

Why others are incorrect:

  • VLANs on the perimeter: VLANs separate network traffic but don't involve address translation.
  • Non-routable address: PAT translates between private (non-routable) and public addresses, but isn't used specifically for non-routable addresses.
  • Servers on the perimeter: While servers might use PAT, it's not their primary purpose.

Beginner Tip: Think of PAT like an apartment building's mailroom - one street address (public IP) with many apartment numbers (port numbers) to route mail to the right recipient.

Question 5

A. EIGRP

B. BGP

C. RIP

D. OSPF

Show Answer & Explanation ▶

Correct Answer: A. EIGRP

Why this is correct: Enhanced Interior Gateway Routing Protocol (EIGRP) is the only option that supports both unequal cost load balancing (via variance command) and uses port 88 for its protocol communications. It's a Cisco proprietary advanced distance-vector protocol.

Why others are incorrect:

  • RIP: Uses UDP port 520 and only equal-cost paths
  • OSPF: Uses IP protocol 89 and only equal-cost paths

Beginner Tip: Think of EIGRP like a smart GPS that can use both highways and backroads (unequal paths) to balance traffic, while others only use identical roads.

Question 7

Which of the following does OSPF use to communicate routing updates?

A. Unicast

B. Anycast

C. Multicast

D. Broadcast

Show Answer & Explanation ▶

Correct Answer: C. Multicast

Why this is correct: Open Shortest Path First (OSPF) uses multicast addresses 224.0.0.5 (all OSPF routers) and 224.0.0.6 (DR/BDR routers) to efficiently share routing updates only with OSPF-enabled devices.

Why others are incorrect:

  • Unicast: Would require individual updates to each router (inefficient)
  • Anycast: Used for delivering content to nearest receiver, not routing protocols
  • Broadcast: Would flood all devices, not just OSPF routers

Beginner Tip: Think of OSPF multicast like a radio station - only devices tuned to that frequency (multicast group) will receive the updates.

Question 11

Correct Answer: A. Check to see if the end connections were wrapped in copper tape before terminating

Question 13

A. Jumbo frames

B. 802.1Q tagging

C. Native VLAN

D. Link aggregation

Show Answer & Explanation ▶

Correct Answer: B. 802.1Q tagging

Why this is correct: 802.1Q VLAN tagging allows a single physical link between switches to carry traffic from multiple VLANs (virtual networks) by adding a tag to each frame identifying its VLAN.

Why others are incorrect:

  • Link aggregation: Combines multiple links but doesn't handle multiple networks

Beginner Tip: Think of 802.1Q like a postal sorting system - tags are like ZIP codes that ensure packages (frames) get to the right neighborhood (VLAN) even when traveling on the same truck (trunk link).

  • Private address space: Being private is fine and does not prevent routing.

Question 23

D. RIP

Show Answer & Explanation ▶

Correct Answer: C. OSPF
